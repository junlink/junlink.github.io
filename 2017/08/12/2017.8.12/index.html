Java 新特性

今天针对java新的一些特型 和改善了很多原先的问题 也简化了很多的步骤

可变参数

    对于不知道方法要传递多少个参数时使用
    在没有该特性产生时 先前都是向方法中传入一个数组 数组是封装的参数的 

    从JDK1.5开始 提供了可变参数的概念 
    示例：
---
public static int add(int ...data){
    .....
    return 0:
}
---

foreach循环

一种加强型的for循环 用于简化数组和集合数据输出的操作
我们来看一下具体的代码
---
for（int x：data）{
    System.out.println(x+" ");
}
---

泛型

泛型用于解决方法参数的统一问题（也可以用向上转型实现） 
---
public class GenericTest {

      public static void main(String[] args) {

           Box<String> name = new Box<String>("corn"); //此处决定 泛型T的类型 也可更改为其他类型 但必须是引用类型（Integer之类）
           System.out.println("name:" + name.getData());
      }

    }

class Box<T> {

    private T data;
    public Box() {

        }

    public Box(T data) {
        this.data = data;
        }

    public T getData() {
        return data;
        }

}
---

枚举

枚举其实实现的就是多例设计模式 
---

package enum_test;

public class TestDemo {
    public static void main(String args[]) {
        Color red=Color.RED;
        System.out.println("枚举的对象序号："+red.ordinal());
        System.out.println("枚举的对象名称："+red.name());
        for(Color c:Color.values()){
            System.out.println(c.ordinal()+"-"+c.name());
        }
    }
}
enum Color {
    RED, GREEN, BLUE;
}
